# 🎯 AI PPT 工作室

一键生成高质量 PPT 和短视频。无需注册，开箱即用。

## ⚡ 三步开始

### 1. 安装（只需一次）
双击 `一键安装.command`

### 2. 开始使用
双击桌面的 `AI PPT 工作室.command`

### 3. 选择模式
```
  1. 📄 PPT 生成（大纲→审稿→HTML+PPTX）
  2. 🎬 视频生成（配音+BGM+字幕→MP4）
  3. 📋 大纲生成（纯内容策划）
  4. 🚀 一键全出（大纲→PPT→视频）
```

## 📦 产出
- `output/slides.html` — 可编辑交互 PPT
- `output/slides.pptx` — PowerPoint 可编辑
- `output/slides_proj/renders/*.mp4` — 1080p 视频

## 🛠 系统要求
- macOS / Windows / Linux
- Python 3.9+
- 视频模式额外需要 Node.js + FFmpeg
